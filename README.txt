# Bluetooth-Controlled RC Car Using Arduino Uno

## Abstract
Brief description of the project.

## Features
- Wireless Bluetooth control
- Forward, backward, left, and right movement
- Smartphone-based operation

### Input Commands

- F → Move Forward
- B → Move Backward
- L → Turn Left
- R → Turn Right
- S → Stop

---

## Tools and Technologies

### Hardware
- Arduino Uno
- HC-05 Bluetooth Module
- L298N Motor Driver
- DC Geared Motors
- Any Hard Cardboard
- Wheels
- Battery Pack
- Jumper Wires

## Software Requirements
- Arduino IDE
- Bluetooth RC Car Android App

## Methods

The implementation of this project consists of the following stages:

1. Assembly of the robot chassis and mechanical components.
2. Installation of DC motors and wheels.
3. Connection of the HC-05 Bluetooth module to the Arduino Uno.
4. Integration of the L298N motor driver module.
5. Programming the Arduino using the Arduino IDE.
6. Establishing Bluetooth communication with the smartphone.
7. Processing control commands and driving the motors.
8. Testing and optimizing the vehicle movement.

## How to Run This Project?

### Step 1: Hardware Assembly
- Assemble the robot chassis.
- Connect the motors to the L298N motor driver.
- Connect the HC-05 Bluetooth module to the Arduino Uno.
- Connect the battery supply.

### Step 2: Software Installation
- Install the Arduino IDE.
- Open the source code provided in this repository.

### Step 3: Upload the Program
- Connect the Arduino Uno to the computer.
- Select the correct board and COM port.
- Upload the source code.

### Step 4: Connect Bluetooth
- Pair the HC-05 Bluetooth module with your smartphone.
- Open the Bluetooth RC Controller application.
- Establish the Bluetooth connection.

### Step 5: Operate the RC Car
- Use the application buttons to control the movement of the vehicle.
